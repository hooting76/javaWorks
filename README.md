 1. 문장(Statement)의 종류: 순차문, 조건문, 반복문, (조건/반복)분기문
    - 순차문 : (순차 실행) 위의 문장 -> 아래 문장 방향으로 진행 (위에서 부터 아래 방향으로 순차 진행)
    - 조건문 : if 와 같은 특정 인자의 값에 대한 조건에 따른 구분
    - 반복문 : for 와 같은 특정 영역에 대한 코드 반복작업을 일으키는 문장
